# 🛒 Full Stack E-Commerce Web Application

A complete **Full Stack E-Commerce Website** built using modern web technologies.  
This application includes all core e-commerce functionalities such as authentication, product management, shopping cart, order processing, admin dashboard, and more.

Note: Online payment integration is not included yet.

---

## 🚀 Live Demo

🔗 Live Site: https://regal-gingersnap-4ab9b8.netlify.app/
🔗 API Server: https://electrohub-project-api.vercel.app/

---

## 📌 Key Features

### 👤 Customer Features
- User Registration & Login (Firebase Authentication)
- Browse Products
- Product Details View
- Add to Cart
- Update Cart Quantity
- Remove Items from Cart
- Place Orders
- View Order History
- Fully Responsive UI

### 🛠️ Admin Features
- Admin Dashboard
- Add / Edit / Delete Products
- Manage Orders
- Manage Users
- Product Stock Control

---

## 🧑‍💻 Tech Stack

### 🎨 Frontend
- HTML5
- CSS3
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React.js
- Firebase Authentication

### ⚙️ Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
